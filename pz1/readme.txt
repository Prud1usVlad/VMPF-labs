 --- Загальнe --- 

    для завантаження роботи на сервер треба: 

        1. залогінитись на ньому (сокор надсилав лист там команда і пароль)
        2. створити папку для файлів пз (наприклад mkdir pz1) 
        3. перейти в папку (cd pz1), створити файли Task1.1.1.sh, Task1.1.2.sh та input.txt та перенести вміст файлів з репозиторію у створені файли на сервері (наприклад nano Task1.1.1.sh, відкривається текстовий редактор і далі копіпаст з файлу)

 --- Завдання 1 ---

    В цьому завданні ми створюємо сайтон файл з якого можна імпортувати функції як в С так і в Пайтон

    Файл Task1.1.1.sh генерує папку, task1 з такою структурою: 

    ./
    ├── build <--- файли що генеруються сайтоном при білді
    │   └── temp.linux-x86_64-3.7
    │       └── hello.o
    ├── hello.c  <--- файл на С створений сайтоном
    ├── hello.cpython-37m-x86_64-linux-gnu.so <--- файл що генерується сайтоном при білді
    ├── hello.pyx <--- сайтон файл з функцією
    ├── setup.py <--- файл що білдить проект
    └── test.py <--- файл на пайтоні що тестує отриману функцію

    для того щоб виконати команди з цього файлу треба:
        
        1. зробити його виконуваним: chmod +x Task1.1.1.sh (тільки при першому виконанні)
        2. викликати його за допомогою: ./Task1.1.1.sh
        3. далі він покроково виконується, можна просто проклікати [Enter] 
        4. результат виконання функції виведеться після тексту "Виконаємо скрипт test.py"
        5. вхідні дані беруться з файлу input.txt (перевірити/змінити вміст можна за допомогою nano input.txt)

 --- Завдання 2 ---

    В цьому завданні ми створюємо сайтон файл з якого можна імпортувати функції з бібліотеки С в Пайтон

    Файл Task1.1.2.sh генерує папку, task1 з такою структурою: 

    .
    ├── build <--- файли що генеруються сайтоном при білді
    │   └── temp.linux-x86_64-3.7
    │       └── pyexamples.o
    ├── lib <--- файли які є результатом створення бібліотеки C
    │   ├── examples.c
    │   ├── examples.h
    │   ├── examples.o
    │   ├── libexamples.a
    │   └── Makefile
    ├── Makefile <--- Мakefile що компілює бібліотеку С
    ├── pyexamples.c <--- файл на С створений сайтоном
    ├── pyexamples.cpython-37m-x86_64-linux-gnu.so
    ├── pyexamples.pyx <--- сайтон файл що робить обгортку для функції з бібліотеки
    └── setup.py <--- файл на пайтоні що тестує отриману функцію

    для того щоб виконати команди з цього файлу треба:
        
        1. зробити його виконуваним: chmod +x Task1.1.2.sh (тільки при першому виконанні)
        2. викликати його за допомогою: ./Task1.1.2.sh
        3. далі він покроково виконується, можна просто проклікати [Enter] (при повторному виконанні запитує чи видалити існуючу папку, потрібно підтвердити бо діл працювати не буде) 
        4. результат виконання функції виведеться після тексту "Виконаємо створений файл main.py"
        5. вхідні дані беруться з файлу input.txt (перевірити/змінити вміст можна за допомогою nano input.txt)
