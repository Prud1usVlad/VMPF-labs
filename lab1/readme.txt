  ---  Попередньо ---

  Для виконання скрипту зі створення проекту, необхідно мати:
    - WSL,
    - встановлений на WSL python
    - встановлений у пайтон модуть virtualenv, найпростіше встановити за допомогою команди: pip install virtualenv

  ---  Перший запуск ---

  1. Для запуску необхідно перейти в директорію root у WSL
  2. За бажанням створити окрему директорію у вигляді ../root/mydir
  3. Скопіювати та перенести у директорію файл generate.sh або створити новий файл і перекопіювати вміст.
  4. Зробити його виконуваним за допомогою команди chmod +x generate.sh
  5. Далі запуск генерації проекту за допомогою ./generate.sh
  6. Під час запуску, запропонують створити нового юзера, дані якого можуть бути використані для доступу до адмінки. 

  Цей комплекс дій, створює повноцінний проект на джанго, та запускає сервер за адресою localhost:7000, або інший порт за необхідності. Увесь необхідний функціонал вже буде впроваджено.

  При переході на адресу http://localhost:7000/ відкривається головна сторінка блогу, вона буде пуста.

  При переході на адресу http://localhost:7000/admin/blog/post/ відкривається адмінка, для входу в який треба використати дані створеного користувача

  на цій сторінці можна переглянути та редагувати всі сутності, зокрема: 

    Tags - теги, має два поля, назва та колір у форматі HEX
    Posts - пости блогу, де є назва, тіло, користувач, та набір тегів.

  Для відображення на сторінці потрібно додати хоча б один пост.

  Пошук по постам здійснюється після введення даних у інпути та натискання кнопки. 

  ---  Наступні запуски ---

    Є два варіанти:

        - запуск без змін, просто запустити файл ./generate.sh та обрати спочатку пункт не видаляти проект, а потім запустити сервер. Таким чином просто запуститься додаток.

        - запуск зі змінами у sh файлі, в цьому випадку треба обрати пункт видалити проект, та ще раз запустити файл. Це призведе до втрати даних з БД.

        - також при потребі можна редагувати все у вже створених файлах, і спробувати запускати як у 1 варіанті, але це не тестувалося.
